# 问题分析

通过分析代码，我发现了几个导致错误的问题：

1. **borderSize 变量初始化问题**：`borderSize` 变量在第 24 行被定义为 `@export var borderSize = 0:`，但没有为其提供默认值的声明，只有一个 setter 函数。当 `_create_grid()` 函数调用 `add_theme_constant_override` 函数时，`borderSize` 变量的值为 `nil`，而该函数需要一个整数类型的参数。

2. **节点实例化问题**：代码在第 41 行仍然使用了 `gridCellNode.duplicate()`，这是之前已经修复过的问题，但现在又出现了。

3. **编辑器模式下的更新机制缺失**：代码中没有 `_ready()` 函数和 `_process()` 函数，这些是之前添加的，用于确保在编辑器模式下正确创建网格。

4. **网格创建状态跟踪缺失**：代码中没有 `grid_created` 变量，用于跟踪网格创建状态。

# 解决方案

1. **修复 borderSize 变量初始化**：确保 `borderSize` 变量有一个默认的整数值。

2. **修复节点实例化问题**：移除 `duplicate()` 调用，直接添加实例化的节点。

3. **恢复编辑器模式下的更新机制**：添加 `_ready()` 函数和 `_process()` 函数，确保在编辑器模式下正确创建网格。

4. **添加网格创建状态跟踪**：添加 `grid_created` 变量，用于跟踪网格创建状态。

# 具体修改步骤
